# Ferramenta de Simulação de Investimentos em Excel

Este repositório apresenta uma ferramenta desenvolvida em Microsoft Excel para simulação e controle de investimentos, criada como parte de um desafio prático de aprendizagem.

## Objetivo do Projeto
O objetivo deste projeto é permitir que o usuário simule investimentos mensais ao longo do tempo, visualizando o crescimento do patrimônio e a geração de dividendos, de forma simples e intuitiva.


## Estrutura do Repositório

- `README.md` → Documentação do projeto  
- `Desafio-simulado-investimento-Curso- Excel-DIO-Carmen Irene Hijar.xlsx` → Planilha de simulação de investimentos  

##  Funcionamento da Ferramenta

A planilha permite ao usuário:

- Informar o salário e o percentual sugerido de investimento
- Definir o valor investido mensalmente
- Simular investimentos ao longo de diferentes períodos (2, 3, 10, 15 e 20 anos)
- Visualizar o patrimônio acumulado e os dividendos mensais
- Analisar cenários de curto, médio e longo prazo
- Distribuir o investimento por tipo de Fundo Imobiliário (Papel, Tijolo, Híbridos e FOFs)
- Visualizar a distribuição dos investimentos por meio de gráfico

Todos os cálculos são realizados automaticamente a partir dos dados inseridos.
